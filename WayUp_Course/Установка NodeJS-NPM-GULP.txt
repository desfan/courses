1. Установить NodeJS

2. Запустить терминал и командами node -v и npm -v (поочередно) проверить версии установленных nodejs и npm. 
Если установились успешно - покажет их версии

3. Командой npm i gulp-cli -g - устанавливаем глобально comand gulp line interface (командная строка gulp)

4. Перейти в папку проекта

5. Командой npm init -y инициализируем проект и, тем самым, создадим файл package.json. Данный файл позволяет переносить
	сборку проекта с места на место (из одного проекта в другой) и в него попадают все основные модули.

6. Далее уже в папке с проектом установить Gulp локально: npm i gulp

7. В папке с проектом создать файл gulpfile.js //В него будут записываться все таски и пайпы

8. Устанавливаем остальные модули:

- устанавливаем gulp-concat: npm i gulp-concat -D (соединяет все файлы, например css или js, в один файл (Конкотинация)). 
- в gulpfile.js пишем: 
	
	const concat = require('gulp-concat');  //объявляется константа concat (const concat) 
						//и подключается модуль по его названию (require('gulp-concat'))

	* Concat не нужен при работе с препроцессорами, т.к. с помощью препроцессора можно подключать файлы 
	с помощью директивы @import, импортируя scss и sass файлы. Все импортированные scss и sass файлы 
	могут быть объединены в одном результирующем css файле. Кроме того, любые переменные или миксины, объявленные 
	в импортированном файле, могут использоваться в главном файле.
	Если с помощью concat необходимо объединить другие типы файлов (например файлы скриптов) - необходимо в package.json
	прописать очередность объединения файлов в виде массива:
	
		const jsFiles = [
			'./app/js/some.js'
			'./app/js/other.js'
		];

По сути установка и подключение модулей заключается в следующем:
- загуглить название модуля, например gulp clean css
- перейти по ссылке на сайт https://www.npmjs.com/package/gulp-clean-css
- установить: npm install gulp-clean-css --save-dev
- подключить модуль в gulpfile.js: const cleanCSS = require('gulp-clean-css');
- скопировать и прописать pipe в gulpfile.js: 
	.pipe(cleanCSS({
		level: 2
	}))
